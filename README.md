# 🎬 Movie Recommender System

A content-based Movie Recommender System built using Machine Learning that suggests movies based on user preferences. This project leverages similarity metrics to recommend movies similar to the one selected by the user.

---

## 🚀 Features

- 🔍 Recommend movies based on similarity
- 🎯 Content-based filtering using ML
- 🖼️ Fetch movie posters using TMDB API
- ⚡ Fast and interactive UI using Streamlit
- 📊 Preprocessed dataset for efficient recommendations

---

## 🧠 How It Works

- Movies are represented using features like genres, keywords, cast, and crew
- Text data is combined and vectorized using **CountVectorizer**
- Cosine similarity is computed between movie vectors
- Top similar movies are recommended
